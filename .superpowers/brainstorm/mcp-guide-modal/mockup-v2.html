Como conectar

1
Abra o terminal
No VS Code, terminal do sistema, ou qualquer shell com o Claude Code CLI instalado.
2
Execute o comando abaixo
Registra o servidor MCP com seu token de autenticação.
$ claude mcp add --transport http chatfunnel \ https://mcp-insider.chatfunnel.com.br/mcp \ --header "Authorization: Bearer mcp_live_a1b2...s9t0" \ --header "Content-Type: application/json" \ --header "Accept: application/json, text/event-stream"
3
Pronto!
O Claude Code agora tem acesso às tools do seu ChatFunnel. Experimente perguntar sobre seus contatos, automações ou funis.

O token não expira por padrão. Você pode revogá-lo a qualquer momento. O header Account-Selected não é necessário — a conta é detectada automaticamente pelo token.
