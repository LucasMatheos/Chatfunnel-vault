Mockup: Novo OrganizationForm (criar organizacao)

Abordagem: como estruturar o form?
Recomendado

A) Dialog simples (Step 1 = dados)

Replica o EditOrgDialog como primeiro step. Ao clicar "Proximo", redireciona para Stripe Checkout (como ja faz hoje). O dialog fecha apos submeter dados basicos.

FLUXO:
1. Dialog abre com form (nome, desc, avatar)
2. "Proximo" → vai pra selecao de plano (dentro do dialog ou nova tela)
3. Escolhe plano → redirect Stripe Checkout
4. Volta → cria org automaticamente
Alternativa

B) Wizard multi-step no dialog

Todos os steps dentro do dialog com stepper lateral. Semelhante ao legado mas com UI nova (shadcn + Tailwind). Mais complexo, porem tudo numa tela so.

FLUXO:
1. Dialog grande (90%) com stepper lateral
2. Step 1: dados org / Step 2: plano / Step 3: checkout
3. Step 4: loading / Step 5: sucesso
4. Tudo dentro do dialog
