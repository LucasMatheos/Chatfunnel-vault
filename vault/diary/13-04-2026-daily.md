# Daily — 13-04-2026

## Resumo

Dia dominado pela expansão do ProfileStep do signup: adicionei 13 novas roles e um select de Segmento de Atuação com 22 opções (+ input livre para "Outros"). A feature virou vertical slice end-to-end (Prisma → core → services DTO/handler → front), consolidada num plano do superpowers. Fechei com uma análise de segurança do formulário que revelou falta de `maxlength`/`@MaxLength`, corrigida em front e back. Também criei um script PowerShell (`sync-core.ps1`) para replicar no Windows o build+sync do `@chatfunnel/core` nos consumers, e investiguei os custom fields nativos (IDs fixos `0000-0000-0000-000X`).
